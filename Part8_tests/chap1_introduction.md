# Cours d'Introduction à Pytest

## Installation

Soit vous travaillez avec Anaconda, soit vous utilisez l'environnement proposé Docker.

Créez la structure des dossiers dans le dossier work de votre Anaconda ou Docker.

```txt
.
├── app.py
├── notebooks
│   └── main.ipynb
├── src
│   └── helpers
│       ├── __init__.py
│       └── utils.py
└── tests
    ├── __init__.py
    └── test_reverse.py
```

On fait un premier test pour voir si tout est bien installé.
